Laboratorio Práctico 1: Estructuras Secuenciales

Documentación de la resolución de 5 problemas básicos (Estructura Secuencial) con Pseudocódigo, Diagramas de Flujo y Codificación en C.

Objetivo y Metodología de la Asignación

Este laboratorio se enfocó en el desarrollo de la lógica algorítmica y la aplicación de la **Estructura Secuencial**. La entrega requirió la documentación completa del proceso de análisis y codificación para cada problema.

Indicaciones de Entrega:

  • Desarrollo de las preguntas en modo **Algoritmo** y **Diagrama de Flujo** (Parte I, 35 Puntos).
  • Sustentación del trabajo en el aula (15%).
  • Puntualidad (15%) y Desarrollo (70%).

3. Cálculo de Precio Neto de Auto Partes

Indicación Real: Una empresa de ventas de partes de autos necesita un programa que calcule y muestre el precio final en balboas de un producto. Para ello, se debe aplicar la siguiente fórmula: $precio\_neto=precio\_costo \cdot (100+margen)/100$.

Nota: Tome en cuenta el precio de costo en balboas y el margen en tanto por ciento.

Pseudocódigo

INICIO LEER PrecioCosto LEER Margen PrecioNeto <- PrecioCosto * (100 + Margen) / 100 IMPRIMIR "Precio Neto:", PrecioNeto FIN

Diagrama de Flujo

Código C

#include <stdio.h> int main() { float precioCosto, margen, precioNeto; printf("Ingrese el precio de costo (en Balboas): "); scanf("%f", &precioCosto); printf("Ingrese el margen de ganancia (en %%): "); scanf("%f", &margen); // Aplicación de la fórmula exacta de la asignación precioNeto = precioCosto * (100.0 + margen) / 100.0; printf("\nEl precio final (neto) es: B/. %.2f\n", precioNeto); return 0; }

4. Geometría del Triángulo Rectángulo

Indicación Real: Escriba un programa que lea los valores de los catetos de un triángulo rectángulo y calcule cuál es la hipotenusa, el área y el perímetro del triángulo mediante las siguientes expresiones:

Fórmulas de la Asignación:

  • Hipotenusa: $h=(c_{1}^{2}+c_{2}^{2})^{1/2}$
  • Perímetro: $p=h+c_{1}+c_{2}$
  • Área: $A=(c_{1}+c_{2})/2$
⚠️ **Nota Importante:** Se mantiene la fórmula de Área $A=(c_{1}+c_{2})/2$ tal como fue especificada en el documento del Laboratorio, aunque la fórmula matemática estándar para el área es $A=(c_{1} \cdot c_{2})/2$.

Pseudocódigo

INICIO LEER C1 LEER C2 Hipotenusa <- RAIZ(C1^2 + C2^2) Area <- (C1 + C2) / 2 // Usando la formula especificada Perimetro <- Hipotenusa + C1 + C2 IMPRIMIR "Hipotenusa:", Hipotenusa IMPRIMIR "Área:", Area IMPRIMIR "Perímetro:", Perimetro FIN

Diagrama de Flujo

Código C

#include <stdio.h> #include <math.h> // Necesario para sqrt() y pow() int main() { float c1, c2, hipotenusa, area, perimetro; printf("Ingrese el valor del cateto 1: "); scanf("%f", &c1); printf("Ingrese el valor del cateto 2: "); scanf("%f", &c2); // Hipotenusa: sqrt(c1^2 + c2^2) hipotenusa = sqrt(pow(c1, 2) + pow(c2, 2)); // ATENCIÓN: Se usa la fórmula de área A=(c1+c2)/2, según la indicación del Laboratorio. area = (c1 + c2) / 2.0; // Perímetro: h + c1 + c2 perimetro = hipotenusa + c1 + c2; printf("\nResultados:\n"); printf("Hipotenusa: %.2f\n", hipotenusa); printf("Área (fórmula provista): %.2f\n", area); printf("Perímetro: %.2f\n", perimetro); return 0; }

5. Informe de Ventas de Refrescos

Indicación Real: Una compañía de refrescos comercializa tres productos: de cola, de naranja y de limón. Se desea realizar un programa que calcule las ventas realizadas de cada producto. Para ellos, se leerá la cantidad vendida (máximo 5000000) y el precio en balboas de cada producto y se mostrará un informe de ventas como el que sigue:

Formato de Informe Solicitado:

"Producto", "Ventas", "Precio", "Total" "Cola", "1000000", "0.17", "170000.00" ...

Pseudocódigo

INICIO LEER Precio_Cola, Precio_Naranja, Precio_Limon LEER Cantidad_Cola, Cantidad_Naranja, Cantidad_Limon Total_Cola <- Cantidad_Cola * Precio_Cola Total_Naranja <- Cantidad_Naranja * Precio_Naranja Total_Limon <- Cantidad_Limon * Precio_Limon Total_General <- Total_Cola + Total_Naranja + Total_Limon IMPRIMIR "Total Cola:", Total_Cola IMPRIMIR "Total Naranja:", Total_Naranja IMPRIMIR "Total Limón:", Total_Limon IMPRIMIR "TOTAL GENERAL:", Total_General FIN

Diagrama de Flujo

Código C

#include <stdio.h> int main() { float p_cola, p_naranja, p_limon; // Usamos long int para la cantidad vendida, respetando el máximo de 5,000,000 long int c_cola, c_naranja, c_limon; float t_cola, t_naranja, t_limon, t_general; // 1. Lectura de Precios printf("--- Ingreso de Precios ---\n"); printf("Precio de Cola (B/.): "); scanf("%f", &p_cola); printf("Precio de Naranja (B/.): "); scanf("%f", &p_naranja); printf("Precio de Limón (B/.): "); scanf("%f", &p_limon); // 2. Lectura de Cantidades printf("\n--- Ingreso de Cantidades Vendidas (Max 5,000,000) ---\n"); printf("Cantidad vendida de Cola: "); scanf("%ld", &c_cola); printf("Cantidad vendida de Naranja: "); scanf("%ld", &c_naranja); printf("Cantidad vendida de Limón: "); scanf("%ld", &c_limon); // 3. Cálculos // Convertimos la cantidad a float para el cálculo, luego volvemos al float para los totales t_cola = (float)c_cola * p_cola; t_naranja = (float)c_naranja * p_naranja; t_limon = (float)c_limon * p_limon; t_general = t_cola + t_naranja + t_limon; // 4. Informe de Ventas (Usando formato de tabla para claridad) printf("\n\n--- INFORME DE VENTAS ---\n"); // %-10s alinea a la izquierda y da 10 espacios printf("%-10s | %-10s | %-8s | %-10s\n", "PRODUCTO", "CANTIDAD", "PRECIO", "TOTAL"); printf("-------------------------------------------\n"); printf("%-10s | %-10ld | %-8.2f | B/. %.2f\n", "Cola", c_cola, p_cola, t_cola); printf("%-10s | %-10ld | %-8.2f | B/. %.2f\n", "Naranja", c_naranja, p_naranja, t_naranja); printf("%-10s | %-10ld | %-8.2f | B/. %.2f\n", "Limón", c_limon, p_limon, t_limon); printf("-------------------------------------------\n"); printf("%-10s | %-10s | %-8s | B/. %.2f\n", "TOTAL", "", "", t_general); return 0; }
