Ejercicio Práctico 1: Estructuras Secuenciales y Condicionales

Documentación de la resolución de problemas enfocados en la aplicación de Estructuras Condicionales y Secuenciales con Pseudocódigo, Diagramas de Flujo y Codificación en C.

Objetivo y Metodología de la Asignación

Este ejercicio práctico introduce la necesidad de utilizar la lógica condicional (IF/ELSE) además de las estructuras secuenciales, para tomar decisiones y presentar resultados específicos.

Criterios de Evaluación:

  • Desarrollo: 70%
  • Sustentación: 15%
  • Puntualidad: 15%

Se solicitó la entrega de Pseudocódigo, Diagrama de Flujo (usando software de ofimática o Dia) y el Código C.

Problemas Resueltos

1. Cálculo de Rendimiento de Viaje

Indicación Real: Escriba un programa que pida el total de kilómetros recorridos, el precio del combustible (diésel, gasolina), el dinero de combustible gastado en el viaje y el tiempo que se ha tardado (en horas y minutos), y que calcule:

  • Consumo de combustible (en litros y dólar) por cada cien kilómetros y/o metros.
  • Consumo de combustible (en litros y dólar) por cada kilómetro y/o metros.
  • Velocidad media (en $km/h$ y $m/s$).

Pseudocódigo

INICIO LEER Kilometros_Totales, Precio_x_Litro, Dinero_Gastado, Horas, Minutos // CÁLCULOS Litros_Gastados <- Dinero_Gastado / Precio_x_Litro Tiempo_Total_H <- Horas + (Minutos / 60.0) // Tiempo total en horas // Consumo Consumo_L_100km <- (Litros_Gastados / Kilometros_Totales) * 100 Consumo_D_100km <- (Dinero_Gastado / Kilometros_Totales) * 100 // Velocidad Velocidad_Km_H <- Kilometros_Totales / Tiempo_Total_H Velocidad_M_S <- (Kilometros_Totales * 1000.0) / (Tiempo_Total_H * 3600.0) // IMPRIMIR RESULTADOS IMPRIMIR "Consumo Litros por 100 km:", Consumo_L_100km IMPRIMIR "Consumo Dólar por 100 km:", Consumo_D_100km IMPRIMIR "Velocidad Media (km/h):", Velocidad_Km_H IMPRIMIR "Velocidad Media (m/s):", Velocidad_M_S FIN

Diagrama de Flujo

Código C

#include <stdio.h> int main() { float km_totales, precio_x_litro, dinero_gastado; int horas, minutos; float litros_gastados, tiempo_total_h; float consumo_l_100km, consumo_d_100km, velocidad_km_h, velocidad_m_s; // Lectura de entradas printf("Ingrese total de kilometros recorridos: "); scanf("%f", &km_totales); printf("Ingrese precio del combustible por litro (B/.): "); scanf("%f", &precio_x_litro); printf("Ingrese dinero de combustible gastado (B/.): "); scanf("%f", &dinero_gastado); printf("Ingrese tiempo tardado (Horas y Minutos):\n"); printf("Horas: "); scanf("%d", &horas); printf("Minutos: "); scanf("%d", &minutos); // 1. Litros gastados litros_gastados = dinero_gastado / precio_x_litro; // 2. Tiempo total en horas (para la velocidad) tiempo_total_h = horas + (minutos / 60.0); // 3. Consumo por cada cien kilómetros (litros y dólar) consumo_l_100km = (litros_gastados / km_totales) * 100.0; consumo_d_100km = (dinero_gastado / km_totales) * 100.0; // Consumo por cada kilómetro (solo imprimir el valor base ya que 100km es suficiente) // float consumo_l_x_km = litros_gastados / km_totales; // float consumo_d_x_km = dinero_gastado / km_totales; // 4. Velocidad media velocidad_km_h = km_totales / tiempo_total_h; velocidad_m_s = (km_totales * 1000.0) / (tiempo_total_h * 3600.0); // Impresión de resultados printf("\n--- Resultados del Viaje ---\n"); printf("Consumo de Litros por 100 km: %.2f L\n", consumo_l_100km); printf("Consumo en Balboas por 100 km: B/. %.2f\n", consumo_d_100km); printf("Velocidad Media: %.2f km/h\n", velocidad_km_h); printf("Velocidad Media: %.2f m/s\n", velocidad_m_s); return 0; }

2. Determinación de Límite de Crédito Excedido

Indicación Real: Desarrolle un código que calcule y muestre el nuevo saldo de un cliente. Si este nuevo saldo excede el límite de crédito permitido, su desarrollo debe mostrar el mensaje: "Límite de crédito excedido". Para un cliente se tienen los siguientes datos: número de cuenta, saldo al inicio del mes, total de todos los artículos cargados por el cliente en el mes, total de todos los créditos aplicados a la cuenta del cliente en el mes, y el límite de crédito permitido.

Pseudocódigo

INICIO LEER Num_Cuenta, Saldo_Inicial, Cargos, Creditos, Limite_Credito Nuevo_Saldo <- Saldo_Inicial + Cargos - Creditos IMPRIMIR "Número de Cuenta:", Num_Cuenta IMPRIMIR "Nuevo Saldo:", Nuevo_Saldo SI Nuevo_Saldo > Limite_Credito ENTONCES IMPRIMIR "¡Límite de crédito excedido!" FIN SI FIN

Diagrama de Flujo

Código C

#include <stdio.h> int main() { int num_cuenta; float saldo_inicial, cargos, creditos, limite_credito, nuevo_saldo; // Lectura de entradas printf("Ingrese el numero de cuenta: "); scanf("%d", &num_cuenta); printf("Ingrese el saldo al inicio del mes: "); scanf("%f", &saldo_inicial); printf("Ingrese el total de cargos del mes: "); scanf("%f", &cargos); printf("Ingrese el total de creditos aplicados: "); scanf("%f", &creditos); printf("Ingrese el limite de credito permitido: "); scanf("%f", &limite_credito); // Cálculo del nuevo saldo nuevo_saldo = saldo_inicial + cargos - creditos; // Mostrar resultados printf("\n--- Resumen de Cuenta ---\n"); printf("Cuenta No. %d\n", num_cuenta); printf("Nuevo Saldo: B/. %.2f\n", nuevo_saldo); // Estructura Condicional: Verificar si excede el límite if (nuevo_saldo > limite_credito) { printf("¡Límite de crédito excedido!\n"); } return 0; }

3. Cálculo de Sueldo Bruto (Horas Extras)

Indicación Real: Desarrolle un código que determine el sueldo bruto para cada empleado (nombre completo, posición). La empresa paga la cuota normal en las primeras 40 horas, y cuota y media en todas las horas trabajadas que excedan de 40. Usted recibe el nombre del empleado, el número de horas que trabajó y la tarifa por hora. Su desarrollo debe recibir como entrada esta información para cada ejecución, para luego determinar y mostrar el sueldo bruto de cada trabajador.

Pseudocódigo

INICIO LEER Nombre_Empleado, Horas_Trabajadas, Tarifa_x_Hora SI Horas_Trabajadas <= 40 ENTONCES Sueldo_Bruto <- Horas_Trabajadas * Tarifa_x_Hora SINO Horas_Extra <- Horas_Trabajadas - 40 Pago_Normal <- 40 * Tarifa_x_Hora Pago_Extra <- Horas_Extra * Tarifa_x_Hora * 1.5 // Cuota y media Sueldo_Bruto <- Pago_Normal + Pago_Extra FIN SI IMPRIMIR "Empleado:", Nombre_Empleado IMPRIMIR "Sueldo Bruto:", Sueldo_Bruto FIN

Diagrama de Flujo

Código C

#include <stdio.h> #include <string.h> int main() { char nombre[100]; float horas_trabajadas, tarifa_x_hora, sueldo_bruto; const int HORAS_NORMALES = 40; const float TARIFA_EXTRA = 1.5; // Lectura de entradas printf("Ingrese nombre completo del empleado: "); // Permite leer nombres con espacios scanf("%[^\n]%*c", nombre); printf("Ingrese horas trabajadas en la semana: "); scanf("%f", &horas_trabajadas); printf("Ingrese tarifa por hora (B/.): "); scanf("%f", &tarifa_x_hora); // Estructura Condicional para calcular el sueldo if (horas_trabajadas <= HORAS_NORMALES) { // Sin horas extras sueldo_bruto = horas_trabajadas * tarifa_x_hora; } else { // Con horas extras float horas_extra = horas_trabajadas - HORAS_NORMALES; float pago_normal = HORAS_NORMALES * tarifa_x_hora; float pago_extra = horas_extra * tarifa_x_hora * TARIFA_EXTRA; sueldo_bruto = pago_normal + pago_extra; } // Mostrar resultados printf("\n--- Reporte de Sueldo ---\n"); printf("Empleado: %s\n", nombre); printf("Horas trabajadas: %.2f\n", horas_trabajadas); printf("Sueldo Bruto Total: B/. %.2f\n", sueldo_bruto); return 0; }

4. Cálculo de Nota Final y Resultado del Curso

Indicación Real: Escriba un programa que permita realizar los cálculos de una nota final de un estudiante universitario. Debe presentar el nombre, las notas por porcentaje, a su vez la final decir sí logra el pase del curso. El pase se logra con una nota de 71 o más.

Nota: La solución se basa en la fórmula de ponderación 40%/30%/30% observada en el desarrollo de su ejercicio, asumiendo que: Parciales (40%), Proyecto (30%), Examen Final (30%).

Pseudocódigo

INICIO LEER Nombre LEER Nota_Parcial, Nota_Proyecto, Nota_Examen // Cálculo de la nota final (Ponderación 40-30-30) Nota_Final <- (Nota_Parcial * 0.40) + (Nota_Proyecto * 0.30) + (Nota_Examen * 0.30) IMPRIMIR "Estudiante: ", Nombre IMPRIMIR "Nota final: ", Nota_Final // Estructura Condicional: Determinar si logra el pase (>= 71) SI Nota_Final >= 71 ENTONCES IMPRIMIR "Resultado: APROBADO" SINO IMPRIMIR "Resultado: REPROBADO" FIN SI FIN

Diagrama de Flujo

Código C

#include <stdio.h> int main() { char nombre[50]; float nota_parcial, nota_proyecto, nota_examen, nota_final; const int NOTA_PASE = 71; // Lectura de entradas printf("Ingrese nombre del estudiante: "); // Leemos el nombre. %*c consume el '\n' remanente scanf("%[^\n]%*c", nombre); printf("Ingrese nota de parciales (40%%): "); scanf("%f", ¬a_parcial); printf("Ingrese nota de proyecto (30%%): "); scanf("%f", ¬a_proyecto); printf("Ingrese nota de examen final (30%%): "); scanf("%f", ¬a_examen); // Cálculo de la nota final (40% + 30% + 30%) nota_final = (nota_parcial * 0.40) + (nota_proyecto * 0.30) + (nota_examen * 0.30); // Impresión de resultados printf("\n--- Reporte de Notas ---\n"); printf("Estudiante: %s\n", nombre); printf("Nota final: %.2f\n", nota_final); // Estructura Condicional: Verificar si aprueba if (nota_final >= NOTA_PASE) { printf("Resultado: APROBADO\n"); } else { printf("Resultado: REPROBADO\n"); } return 0; } }
